# Carcassonne Score Tracker

A simple HTML-based Carcassonne game score tracker.

Created by [Ruslan Makrenko](https://github.com/rmakrenko).

## Features

- Track 1 to 8 players.
- Quickly add or subtract common Carcassonne scoring values.
- Switch between English and Ukrainian.
- Mobile-first portrait layout.
- Save player names, scores, and language in the browser with `localStorage`.

## Publish on Render

This repository includes `render.yaml` for Render static-site publishing.

Manual Render settings:

- Service type: `Static Site`
- Build command: leave empty
- Publish directory: `.`
